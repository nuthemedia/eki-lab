# Project

Route: `/taikyoku`

## Purpose

「易有太極、是生兩儀、兩儀生四象、四象生八卦」を、一続きの3D空間とスクロール操作で体験する導入アプリ。占い結果ではなく、1→2→4→8→64という生成構造を直感的に理解させる。

## Audience

易の予備知識がない一般ユーザー。スマホで縦に進み、短い言葉と立体変化から基本構造を覚えられること。

## MVP

- 太極・両儀・四象・八卦・六十四卦の5段階
- 単一WebGL空間を進むスクロール連動カメラ
- 意味のある対象だけへのタップ・横ドラッグ
- reduced motionとWebGL非対応時の静的表示
- `/hakke`への導線

## Out of scope

占断、個別八卦・六十四卦解説、辞典、音、ログイン、保存、共有、自由カメラ、端末傾き操作。
