# Features

## Core Features

- 陰陽ビルダー: 「陽」「陰」ボタンで爻を必ず下から積む(1本目=下爻、2本目=中爻、3本目=上爻)
- お手本一致判定: お手本と同じ爻だけが定着する。違う爻は静かに溶けて消える(✗・減点・エラー文なし)
- 自然現象ステージ: 3本そろうと、その卦の自然現象が中央 canvas で発生(約2秒)。タップでスキップ
- 結果カード: 形・漢字・読み・属性・自然イメージ一文を段階リビール
- 学習順: 構造コントラスト対 — 乾(全陽)→坤(全陰)→離(中だけ陰)→坎(中だけ陽)→震(下だけ陽)→艮(上だけ陽)→兌(上だけ陰)→巽(下だけ陰)
- 進行ドット: 8個、現在位置と完了を表示
- 想起モード: お手本の三本線を出さず、漢字+読み+属性を手がかりに記憶から作る。同じ爻(スロット)で3回 ghost が溶けたら「そっとお手本」が淡くフェードイン、その爻が埋まると消える
- 進捗永続化: 完走回数(guided/recall)と卦ごとの想起実績(recallBuilt / hinted)を localStorage に記録
- えらぶ: 8卦シャッフル順。ステージで自然現象+完成形を見て、漢字4択(正解1+他の卦3)からえらぶ。違う字は静かに薄れて押せなくなる(✗なし)。合う字で ResultCard が開く
- ずかん: 8卦グリッド+上部ステージ。セルタップでその卦の自然現象を何度でも再生。「もどる」で intro へ。入口は intro メニューと完成画面のセルタップ
- トップメニュー(固定・学習順): 6ステップを常に同じ順で2グループ表示(まなぶ=つくる/おもいだす/えらぶ/きょうのひとつ、ひろげる=ずかん/かさねる)。入れ替えはしない。未解放は淡くロック(dashed・非活性・「まず〜から」)。解放条件は つくる=常時、おもいだす/きょうのひとつ/ずかん=`guidedRounds>0`、えらぶ/かさねる=`recallRounds>0`。「つぎにやること」を1つだけアクセント塗りで強調(guided未→つくる、recall未→おもいだす、choose未→えらぶ、以降→きょうのひとつ)
- 完成画面: guided後→recall / recall後→choose / choose後→recall を主CTAにする交互反復。セルタップで ずかん へ
- いつでもトップに戻る: build/nature/reveal・えらぶ・完成画面すべてに「← トップへ」リンクを常設。確認ダイアログなしで即 intro に戻る(mode の値に関わらず `phase==="intro"` が最優先で評価されるため安全)
- まいにちひとつ: intro メニューの「きょうのひとつ」(guided 完走後に解放)。`getDailyTrigramId()` が perTrigram の実績から最も弱い1卦を選び、その卦だけを想起ビルドさせる。同日中に再訪すると同じ卦が出る(UTC日付基準)。完成で `recordTrigramRecall` が呼ばれ、想起モードと同じ実績に加算される
- かさねる: intro メニューの「かさねる」(recall 完走後に解放)。8卦グリッドから下卦→上卦の順にタップで選び、`hexagramFromLines()` で引いた64卦の名前・読みと構造的な一言(「◯◯の上に、◯◯。」)だけを表示。占断・解説文・64卦辞典への遷移はなし。進捗永続化なし
- サウンド(手続き的 Web Audio、アセットなし): 爻を積む `tap`、溶ける `dissolve`、完成の `chime`、卦ごとの自然音 `nature(id)`×8。全体は chill(マスターに lowpass+手続き的リバーブ+低め gain)、自然音はピンクノイズ+フィルタ+LFO のレイヤーでリアル寄り(単純なビープにしない)。右上のスピーカーアイコンでオン/オフ、**既定オフ**、`hakke-sound-v1` に永続。オフの間は AudioContext を生成せず完全無音

## User Flow

intro(固定メニュー。初回は つくる のみ解放・強調)→「つくる」→ build(お手本を見て陰陽タップ ×3)→ nature(現象)→ reveal(結果カード)→「つぎへ」→ 次の卦 … 8卦目の reveal →「ひらく」→ complete(8卦グリッド)→ 主CTA or「← トップへ」。以降 intro に戻るたび、解放済みステップと「つぎにやること」が進捗に応じて更新される

## States

- build 中の誤タップ: ghost 爻が約0.5秒で溶けて消える。連打しても組み立て中の爻は壊れない
- nature 中のタップ: 即 reveal へスキップ
- reduced-motion: nature は最終フレーム静止画を短く表示して reveal へ
- リロード: 周回の途中経過は保持しない(intro に戻る)。完走回数・卦ごとの実績・きょうの一卦は localStorage(`hakke-progress-v1`)に、サウンド設定は `hakke-sound-v1` に残る
- localStorage 不可(プライベートモード等): 保存に失敗しても黙って続行。メニューは毎回「つくる」のみ解放状態に戻る
- トップへ復帰: nature 再生中に離脱しても canvas の rAF・タイマーはリークしない(NatureStage の unmount クリーンアップで停止)
- サウンド既定オフ: 初回はどのタップでも無音(AudioContext を作らない)。オンにするとその場のタップから発音、リロード後も維持

## Acceptance Criteria

- 8卦すべてを通しで作れる。各卦の完成形・漢字・読み・属性・イメージ文が `data/hakke/trigrams.ts` と一致する
- 兌☱(下から 陽陽陰)・巽☴(下から 陰陽陽)が画面上で正しい向きに描画される
- 誤タップで爻が定着しない・UI にエラー語彙が出ない
- nature 演出がタップでスキップでき、スキップしても reveal に正しく進む
- 想起モードでお手本の三本線が出ない・同じ爻で3回迷うとお手本が淡く浮かぶ・埋まると消える
- 初回 intro は 6ステップが学習順に固定表示され、つくる のみ解放・強調、他5つはロック表示。guided 完走後は おもいだす/きょうのひとつ/ずかん が解放され「つぎにやること」が おもいだす に移る。recall 完走後に えらぶ/かさねる が解放される
- サウンドは既定オフ(トグル未操作では AudioContext を作らず無音)。オンにするとタップ・自然現象・完成で発音し、`hakke-sound-v1` が永続してリロード後も維持。自然音はビープでなくノイズ+フィルタのレイヤーで鳴る
- えらぶで違う字が薄れて押せなくなり、✗・正解・スコア系の表現が出ない
- ずかんのセルタップで現象が再生され、「もどる」で intro に戻れる
- build/えらぶ/完成画面のいずれからも「← トップへ」で intro に戻れる
- 乾+坤を かさねる と「地天泰」になる(TABLE 定義と一致)。6本の爻が下から正しく描画される
- 「きょうのひとつ」を同日中に2回開くと同じ卦が出る
- `npm run lint` と `npm run build` が通る
