# Project

Route: `/hakke`

## Purpose

八卦(はっけ)を普通の人が効率よく覚えるためのスマホ向け学習アプリ。占いアプリではなく、八卦の形・漢字・自然属性・陰陽構造を「自分の手で作る」ことで記憶に定着させる。

## Audience

八卦を知らない・うろ覚えの一般ユーザー。易の予備知識ゼロを前提とする。スマホで片手・数分で触れること。

## MVP Scope

- 陰陽ボタンで、お手本と同じ八卦を下から積んで作るビルダー
- 完成後に形・漢字・読み・属性・短い自然イメージを段階表示する結果カード
- 8つの八卦を「構造コントラスト対」の順(乾→坤→離→坎→震→艮→兌→巽)で学習する流れ
- 進行ドットによる現在位置表示
- 完成した八卦に対応する自然現象を Canvas 2D の中央ステージで再生する演出(約2秒、タップでスキップ可)
- 想起モード「おもいだしてつくる」: お手本なしで漢字+読み+属性を手がかりに作る。同じ爻で3回迷うと「そっとお手本」が淡く浮かぶ
- 学習進捗の localStorage 永続化(`lib/hakkeProgress.ts`)と、完走履歴に応じたモード分岐(初回=お手本つき、再訪=想起や えらぶ を主CTAに)
- えらぶモード: 自然現象と形から漢字4択をえらぶ逆方向練習(シャッフル順)
- ずかん: 8卦一覧から自然現象をリプレイ
- いつでもトップに戻れる「← トップへ」リンク(つくる/おもいだす/えらぶ/完成画面すべてに常設)
- まいにちひとつ: perTrigram の実績(未着手・経過日数・hinted・chooseMissed)から最も弱い1卦を選び、想起ビルドさせる軽量デイリー体験
- かさねる: 下卦・上卦をタップで選び、`hexagramFromLines()` で対応する64卦の名前・読みだけを表示する軽いブリッジ体験(占断・解説は含めない)
- トップメニュー: 6ステップを固定の学習順(まなぶ=つくる/おもいだす/えらぶ/きょうのひとつ、ひろげる=ずかん/かさねる)で常時表示。未解放は淡くロック表示、「つぎにやること」を1つ強調
- サウンド: 手続き的 Web Audio(アセット不要)。全体は chill、自然音はノイズ+フィルタ+LFO でリアル寄り。スピーカーアイコンでオン/オフ(**初回オン**、保存済み設定はlocalStorageで永続)

## 学習ロードマップ

写して作る → 思い出して作る → 逆方向で結ぶ → 間隔をあけて繰り返す、の順:

1. つくる(お手本つき)— 実装済み
2. おもいだしてつくる(想起)— 実装済み
3. えらぶ — 自然現象と形を見て漢字4択からえらぶ逆方向練習。シャッフル順 — 実装済み
4. まいにちひとつ — perTrigram の実績(hinted / chooseMissed / 未着手 / 経過日数)を使った軽い間隔反復 — 実装済み
5. ずかん — 8卦一覧から自然現象リプレイ — 実装済み
6. かさねる — 八卦を2つ重ねて64卦への橋渡し — 実装済み(軽いブリッジ版。名前・読みのみ、占断や64卦辞典への遷移は含まない)

ロードマップ1〜6すべてが実装済み。

## Out Of Scope

- 64卦の本格占断・解説文・iching 辞典への遷移、AI機能、ログイン、課金
- スコア・正解率・ランキング・クイズ表現
- Three.js / WebGL、音声アセット(音は手続き的 Web Audio のみ)
- 自然現象から三本線が逆生成される演出(インターフェースの布石のみ)
- まいにちひとつの正式な間隔反復アルゴリズム(SM-2等)。素朴なスコアリングのみ
- 本番デプロイ・ルート登録(ローカル確認後の別ステップ)
