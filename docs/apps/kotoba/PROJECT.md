# Project

Route: `/kotoba`, `/kotoba/[slug]`

## Purpose

易経の重要な五つの句を、文章より先に動きと操作で理解するインタラクティブ作品集。「易有太極」の続編として、生成構造の次に変化の思想を一つずつ体験させる。

## Audience

易の予備知識がない一般ユーザー。スマートフォンでも、一つの操作と短い日本語から句の中心概念をつかめること。

## Scope

- 五つの作品一覧と個別の静的ルート
- Canvas 2Dによる異なる五つの現象
- 原文、読み下し、現代語訳、解説、出典
- reduced motion、キーボード操作、新規環境音
- 「易有太極」およびeki-lab一覧への導線
